    MCP Tools¶

    Tools exposed by the seller MCP server (ad_seller.interfaces.mcp_server), enumerated from the live FastMCP registry.

    Total: 46 tools

    Tool Description
    approve_or_reject Submit an approval decision. decision: 'approve', 'reject', or 'counter'.
    bulk_deal_operations Process multiple deal operations in one batch.
    create_api_key Create an API key for a buyer or agent.
    create_curated_deal Create a deal with curator overlay. The curator's fee is added on top.
    create_deal_from_template Create a deal directly from parameters (one-step, no quote needed).
    create_package Create a new curated package in the media kit.
    deprecate_deal Mark a deal as deprecated with reason and optional replacement.
    distribute_deal_via_ssp Distribute a deal through configured SSP(s).
    export_deals Export deals in DSP-native format (generic, ttd, dv360, amazon, xandr).
    get_buyer_activity Show buyer agent engagement: who accessed inventory, initiated deals,
    get_config Get current configuration summary (no secrets).
    get_deal_lineage Get the lineage chain for a deal — parents and replacements.
    get_deal_performance Get delivery and performance metrics for a deal.
    get_gam_delivery_report Pull a delivery report from Google Ad Manager for one or more orders.
    get_inbound_queue Get everything waiting for publisher action: pending approvals, unresolved
    get_pricing Calculate tiered pricing for a product based on buyer identity.
    get_rate_card Get the current rate card (base CPMs by inventory type).
    get_setup_status Check what's configured and what's missing. Use this on first connection
    get_supply_chain Get the seller's supply chain transparency info (sellers.json format).
    get_sync_status Check the status of the inventory sync scheduler.
    health_check Check system health — storage, ad server, SSPs.
    list_agents Show the 3-level agent hierarchy and their roles.
    list_api_keys List active API keys.
    list_buyer_agents List registered buyer agents and their trust levels.
    list_configurable_flows Introspect current automation config: approval gates, guard conditions,
    list_curators List registered curators (Agent Range is pre-registered).
    list_gam_orders List orders from Google Ad Manager (GAM).
    list_inventory List raw inventory from the ad server (before product mapping).
    list_orders List orders and their current states.
    list_packages List packages in the media kit. These are what buyers browse.
    list_pending_approvals List pending approval requests waiting for human decision.
    list_products List products in the catalog. These are the inventory items available for deals.
    list_sessions List active buyer conversation sessions.
    list_ssps List configured SSP connectors and routing rules.
    migrate_deal Replace an existing deal with a new one. Creates lineage tracking.
    push_deal_to_buyers Push a deal to buyer endpoints via IAB Deals API v1.0.
    register_buyer_agent Discover and register a buyer agent by URL.
    request_quote Request a non-binding price quote for a product.
    revoke_api_key Revoke an API key.
    set_agent_trust Set trust level for a buyer agent.
    set_approval_gates Configure approval gates. required_flows is comma-separated:
    set_publisher_identity Set the publisher's identity (name, domain, organization ID).
    sync_inventory Trigger inventory sync from the ad server (GAM or FreeWheel).
    transition_order Transition an order to a new state (e.g., draft→approved→delivering).
    troubleshoot_deal Diagnose deal performance issues via SSP diagnostics.
    update_rate_card Update the rate card. Pass entries as JSON array:
